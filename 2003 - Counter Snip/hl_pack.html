Les Packs HL²

Les Pack Half Life ²
Je vais essayer dans cette rubrique de vous expliquer les différents moyen de se procurer HL² et les différentes versions qui vont sortir.
Bien sûr, toutes ces informations ont été grignotés sur le net et se sont en aucun cas de source officielle.

Il y aura apparament 5 manières de se procurer HL² :



Version "Simple Joueur" :
Ce pack fournira le jeu HL² tout simple sans mode Multijoueur ni MOD.
En d'autres termes, cette version du jeu est pour les gamers occasionnels n'ont pas d'intérêt à télécharger des nouveaux mods.

 



Version "Standart" :
C'est la version standart de HL² que la plupart des gamers voudront acheter. Cette version permettra de jouer Online ainsi qu'au nombreux Mod qui vont sortir.
Ce pack coutera environ 60€ soit le prix d'un bon jeu.

 



Version "Collector" :
Ce sera probablement dans les 80€ environ pour les HardCore Gamers. Il viendra probablement dans une certaine caisse de fantaisie avec surement un ou plusieurs bouquins : guides stratégiques.

 

Version Steam :
Si vous êtes trop paresseux pour aller au magasin, vous pouvez pouvoir obtenir HL² en le téléchargeant par Steam soit en payant le jeu au prix normal soit en payant un abonnement qui vous permettra de télécharger d'autres Mods.


Pack ATI :
Half Life² sera sourni avec certaines cartes ATI tels que la 3D Prophet 9600 et 9800.
Dans ce cas, le jeu est gratuit !
