    Servers CS .:. Comment se connecter ?

    Pour se connecter à un server, il y a 2 manière :

    1a°) Dans Counter Strike 1.5 :

    - Lances Counter Strike
    - Cliques sur [ Play CS ] [ Multijoueur ]
    - Cliques sur [ Internet Games ] [ Partie Internet ]
    - Cliques sur [ Filter ] [ Filtre ]
    - Coches[ Sont dans ma liste de Favoris ]
    - Valides en cliquant sur [ Filter ] [ Filtre ]
    - Cliques sur [ Add Server ] [ Ajouter un server ]
    - Tapes l'adresse du server + OK
    - Et voilà !

    1b°) Dans Counter Strike 1.6 :

    - Lances Steam
    - Sélectionnes SERVERS
    - Ouvres l'onglet Favoris
    - Click avec le bouton droit au milieu de la fenêtre
    - Sélectionnes "Add server by IP adress"
    - Entres l'adresse voulu + OK
    - Double click sur ce server

    2°) Avec un logiciel :

    - Télécharges ce petit logiciel : HL Fast Connector 2 ( 20ko )
    - Lors de la primière utilisattion, cliquez sur Config et montres lui où se trouve HL.
    - Sélectionnes "cstrike" dans la ligne MOD
    - Tapes l'adresse puis cliques sur RUN.
    - Et voilà, C'est tout.
