CS 1.6 Offline + Bots !

II°) Installation :

Exécutes le fichier "cs16full_v4+zbot.exe"
Click sur "..... >" plusieurs fois
Sélectionnes le chemin d'install que tu veux
( mais n'écrase pas le vrai cs 1.6 de préférence ! )
Une fois l'install terminé, lances le jeu par l'icone créee sur le bureau.

II°) Configuration :

Une fois le jeu lancé, cliquez sur Options :

Keyboards : Dans cet onglet, tu peux paramettrer tes touches.
IMPORANT : Mets une touche à " Activate in-game GUI"

Mouse : Une chose intéréssante, la sensibilité de la souris.

Audio : On passe

Vidéo : Si vous avez un assez bon PC, paramétrez comme ceci :

- Renderer : OpenGL
- Résolution : Environ 1152*860
- Color Quality : 32

Voice : Tout au Max

Lock : Rien

Multiplayer : Votre Pseudo

III°) Jouer :

Pour lancer une partie en LAN avec des Bots, cliquez sur "New Game", sélectionnez ensuite votre map puis OK !

C'est tout simple !

III°) Les Bots :

Voici la partie qui peut être la plus compliquée, mais quand même très simple !

Pour ajouter des Bots :

1°) Vous appuyer sur la touche configurée à "Active in-game menu" et vous sélectionnez : Add Zbot -> Quick Add Bot

2°) Dans la console, tappez : bot_add

Changer le niveau des bots :

Activez le Menu In-Game, sélectionnez Add ZBOT / Set zBot puis choississez le niveau.
Commande = bot_difficulty x ( 0=Nul, 3 = Expert )

Tuer tous les Bots ( équivalent = 3 chez les PodBots )

Activez le Menu In-Game, sélectionnez ZBOT MISC -> End Round
Commande = bot_kill

Je vous laisse dévinez vous même deviner à quoi servent les commandes du menu !

Commandes Console :

bot_add_t : Ajoute un TERRO
bot_add_ct : Ajoute un Contre - Terroriste

bot_kick : Kick un bot

bot_knives only : Couteau seulement
bot_pistols_only : Pistolet seulement
bot_snipers_only : Snipers seulement
bot_all_weapons : Toutes les armes

bot_allow_pistols [1-0] : Autorise ou non les Pistolets
bot_allow_shotguns [1-0] : Autorise ou non les Pompes
bot_allow_sub_machine_guns [1-0] : Autorise ou non les Mitraillettes
bot_allow_rifles [1-0] : Autorise ou non les Fusils
bot_allow_machine_guns [1-0] : Autorise ou non la Machin GUN
bot_allow_grenades [1-0] : Autorise ou non les Grenades
bot_allow_snipers [1-0] : Autorise ou non les Snipers
bot_allow_shield [1-0] : Autorise ou non le Bouclier
