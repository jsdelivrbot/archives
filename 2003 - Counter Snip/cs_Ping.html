Comment améliorer son ping :

C koi ce truc : le ping ?

Le ping (aussi appelé latency) est le temps que met une information pour aller du serveur de jeu jusqu'à votre ordinateur. Pour ceux qui n'auraient pas encore compris : plus le ping est élevé, plus les informations (comme les balles, impacts, etc..) mettront du temps à vous parvenir; plus vous aurez l'impression de ça ramme ( On dit que sa LAG )
Si votre ping est bas, la qualité de jeu est meilleure ( comme dans les réseaux où le ping est quasiment nul )

Pour une qualité de jeu satisfaisante, un ping inférieur à 200 / 250 est préférable; après, le jeu devient très difficile, avec de nombreux 'bugs' qui sont justement dus à ce ping élevé..

Sachez qu'avec l'ADSL, votre ping moyen doit tourner autour de 80 / 100
Avec le câble, cela dépend des opérateurs, mais le ping est bien inférieur à 100 (normalement meilleur que l'adsl)
Avec un modem RTC (56 Ko/s), le ping peut aller de 150 à 400; ça dépend aussi du server
Plus étonnant, avec une ligne Numéris ( 64kp/s ) , vous aurez un meilleur ping qu'avec l'ADSL.

Y-a-t'il d'autres Astuces pour avoir un meilleur ping ?

D'abord, sachez que la connexion par l'opérateur Internet AOL (ADSL et RTC) est d'assez mauvaise qualité en général; faites-y donc attention si vous voulez changer d'opérateur...
D'autres paramètres rentrent en compte comme la distance vous séparant de votre opérateur, la qualité de votre ligne, la localisation du server de jeu ( plus il est loins, plus vous aurez de ping ! )

Pour optimiser tout sa, déjà, désactiver votre Firewall ( ou tout autre proxy ) car ceux ci utilisent des ressources.

Lorsque vous choississez le server sur lequel vous allez jouer, prenez en un qui répond rapidement.

Sachez aussi que moins un server a de place, moins il y aura de ping.
