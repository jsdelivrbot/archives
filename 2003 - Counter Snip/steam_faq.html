Pleins de Questions = Pleins de Réponses !

Questions/Réponses sur les problèmes rencontrés :  

Pourquoi, quand je lance Steam, soit il se lance après quelques minutes, soit j’ai un message d’erreur (Could not connect to Steam network …) ?
-> Le réseau Steam network est probablement encombré (vous pouvez suivre sont évolution ici) … il faut simplement attendre, relancer plusieurs fois Steam ou redémarrer (ça fonctionne de temps en temps ). En clair : PATIENCE !!!  

Pourquoi, quand je suis mort, je ne peux plus ni parler, ni chatter, ni changer les vue ?
-> Aller dans votre config.cfg se trouvant dans ...Steam\steamApps\XXXX@XXX.XX\counter-strike\cstrike remplacer la ligne spec_autodirector_internal '1' 0 = pas bon 1 = bon
On peut débloquer en cours de partie en tapant dans la console : "+duck" et ensuite "-duck"...

Comment faire pour avoir le nom des joueurs centrés a l'écran ?
-> Il faut avoir dans le config.cfg hud_centerid "1"

J’ai des chutes de FPS, surtout sur la nouvelle carte Aztec … pourquoi ?
-> le rajout des effets climatiques sur cette carte provoque ces chutes, pour enlever ces effets : mettre cl_weather à 0 dans le config.cfg

Remettre le nom des Armes et anciennes Polices :
-> La solution est décrite ici (Merci Foul et Toma)

Même si j’ai tout bien installé, et que j’ai déjà pu jouer, quand je lance Steam, j’ai un message " Login to steam failed. Either the specified account does not exist, or the password was wrong. Please check your email address and password and try again. " ou "counter strike will use 350 MB on disk. Steam will now acquire (download) will of the required content" … que dois-je faire ???
-> Ces deux messages sont faux, ils sont dus à une surcharge du serveur d'authentification, donc même conseil que pour le point ci dessus …

Pourquoi il n’y a plus de Bots dans cette version de CS alors qu’il y en avait dans la beta ?
-> Valve et la société Turtle Rock Studios sont entrains de peaufiner une nouvelle version des bots officiel, ils réapparaîtront dans une prochaine mise à jour …

Depuis quelques jours je n’arrive pas à lancer la partie Friends de Steam, c’est
normal ?
-> Non, c’est simplement que le service friends est down pour quelques jours, pour maintenance

Je tourne sous Windows 98 et j’ai beaucoup de mal à installer ou lancer Steam … que dois-je faire ?
-> Même si Steam est compatible avec Windows 98 et ME, il est extrêmement instable sous ces 2 OS … Il n’y a pas de solution pour l’instant.

Sera-t-il possible de jouer sans être connecté à internet (pendant les lans, …) ?
-> pour l'instant c'est impossible mais une MAJ de steam, prévue dans quelques semaines va normalement permettre de jouer sans passer par le net ...

Que se passera t il avec les serveurs d’identification Won dans les prochains jours ???
-> Valve ne coupera pas les serveurs WON tant que Steam ne fonctionnera pas de manière optimale, et d'ici là vous pouvez continuer de jouer à CS1.5.
