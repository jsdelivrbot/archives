.:. Créer ses tags .:.

Ce tutorial va vous permettre de créer manuellement des tags en couleur. Pour plus de facilité, vous pouvez utiliser le programme Half Life Logo Créator , qui crée un tag automatiquement à partir d'une image.

Dans Half-Life (et donc Counter-Strike), il existe deux types de tags :

- Les tags par défaut, monochromes et de taille fixe.
- Les tags en couleur et de dimensions variables.
(Le nombre de couleurs et les dimensions autorisées n'est cependant pas illimité.)


.:. Comment sa marche ? .:.

Le logo (également appelés spray en anglais) est envoyé de votre PC sur le serveur sur lequel vous jouez. Ensuite, le serveur l'envoie aux différents joueurs pour qu'ils puissent l'afficher.

La taille maximale du fichier pldecal.wad est de 15kb. Vous pouvez donc utiliser les tailles que vous voulez sachant que la taille maximum (en pixels) est de 144*80 et la recommandée est de 128*64.



.:. Créer son tag .:.

La première étape dans la création d'un tag est de créer un fichier Image BMP respectant les règles suivantes :

  • 256 couleurs
  • Largeur : multiple de 16 pixels.
  • Hauteur : multiple de 16 pixels.
  • Le produit (Largeur x Hauteur) doit être strictement inférieur à 12288 pixels.
  • La taille de l'image de doit pas dépasser 15Ko.

C'est l'étape la plus compliqué, vous comprendrez pourquoi.



.:. La compilation .:.

1° Installez le logiciel "Half-life Logo Creator" sur votre disque-dur
2° Lancez l' application "HLC.exe"

3° Parcourez votre disque-dur à la recherche de votre image
4° Sélectionnez l' image *.bmp qui est votre tag
5° Cliquez sur le bouton "Load bitmap" (normalement, une image sera afficher tout à droite)

6° Avec le paramètre "Image width" (= largeur de l' image), sélectionner la largeur de votre image
7° Avec le paramètre "Image height" (= hauteur de l' image), sélectionner la hauteur de votre image

8° Cliquez sur le bouton "Locate Halflife"
9° Aggrandissez ou maximisez la fenêtre "LOCATE HL.EXE" pour voir apparaître le bouton "Ok"
10° Indiquez le répertoire de l' executable hl.exe (généralement c:\SIERRA\Half-life\hl.exe)
11° Validez avec "Ok"

12° Avec le choix "Make a color logo for mod", le mod Counter Strike
13° Vérifiez que tous les paramètres sont complètés
14° Cliquez sur le bouton "Make WAD".

C'est fini ! Maintenant, allez dans CS tester votre chef d'oeuvre



.:. IMPORTANT .:.

Pour tester le tag, surtout ne pas aller dans le menu "Customize" ("Personnaliser") de half-life.
Cet écran affiche bien des tags, mais uniquement les tags monochrome, et votre pldecal.wad sera écrasé par le logo par défaut de Half-Life.
