Comment activer le JavaScript :

 

FireFox Internet Explorer Konqueror Nescape Mozilla

 

 

 

 

Tutorial rédigé par YoTsumi pour Ix-Team

 

 

 

 

 

 

Activer les JavaScript sous FireFox :

  • Menu Outils, Options
  • Sélectionnez «Fonctionnalités Web»
  • Cochez la case : «Javascript activé»
  • Vérifiez que toutes les cases sous le bouton «Avancé» sont cochées.
  • Valider en cliquant sur «OK».
  • Fermer la fenêtre par «OK».

Activer le JavaScript sous Internet Explorer :

  • Menu Outils, Options Internet
  • Sélectionnez l'onglet «Sécurité».
  • En haut de la fenêtre, choisissez l'icône «Zone Internet».
  • En bas de la fenêtre, cliquez sur le bouton «Personnaliser le niveau».
  • Descendez jusqu'à la rubrique «Script».
  • Sous «Active Scripting», cocher la case «Activer».
  • Cliquez sur «OK» et encore sur «OK».

Activer le JavaScript sous Konqueror :

  • Menu Configuration, Configurer Konqueror
  • Ouvrir l'onglet «Java et Javascript»
  • Puis l'onglet «Javascript»
  • Cocher «Toujours activer»
  • Validez par «OK»

Activer le JavaScript sous Nescape :

  • Menu Edition, Préférences
  • Double-cliquez sur «Avancé» pour afficher la liste.
  • Double-cliquez sur «Scripts & Modules externes».
  • Sous «Activer Javascript» cocher la case «Navigator».
  • Cochez toutes les cases sous «Autoriser les pages web à ».
  • Cliquez sur «Ok».

Activer le JavaScript sous Mozilla :

  • Menu Edition, Préférences
  • Rubrique «Avancées», sous-rubrique «Scripts et modules externes»
  • Sous «Activer le Javascript» cochez la case «pour le navigateur».
  • Vérifiez que chaque case dans le cadre en dessous est cochée.
  • Validez en cliquant sur «OK».
