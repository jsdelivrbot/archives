  VW New Beetle Dune : pour déverrouiller la peinture personnalisée, les amateurs doivent se qualifier parmi les trois premières places dans toutes les courses sur circuit de San Francisco. Les professionnels doivent terminer quant à eux premiers dans toutes ces courses.

  VW New Beetle RSi : il exite trois peintures personnalisées pour la RSi. Pour déverrouiller la peinture personnalisé Microsoft Cup, vous devez réussir au premier partiel de la session d'entraînement de Londres.

  New Mini Cooper : pour déverrouiller la peinture personnalisée, les amateurs doivent se qualifier parmi les trois premières places dans toutes les courses éclair de Londres. Les professionnels doivent terminer quant à eux premiers dans la moitié de ces courses.

  Ford Mustang Fastback 1968 : pour déverrouiller la peinture personnalisée, vous devez passer avec succès le 3e partiel de la course éclair de San Francisco.
